Role: Machine Learning Engineer (New Grad / Entry Level)

Company: QuantumLeap AI

This role is ideal for a strong new grad with hands-on project experience in ML, deep learning, and data pipelines.

- Conduct A/B testing and statistical analysis to validate model performance in production
- Optimize model inference latency and throughput for real-time serving
- Collaborate with data engineers to build scalable ETL pipelines using Apache Spark and Kafka
- Write clean, tested, production-quality Python code with proper logging and error handling

- Understanding of statistical concepts: hypothesis testing, confidence intervals, A/B testing
- Familiarity with SQL and relational databases (PostgreSQL preferred)
